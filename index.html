About Me

I am a Computer Engineering graduate student at Stony Brook University. I joined this university in Fall 2017 and expect to graduate in December 2018.I am currently interning at IPC Systems in the Research and Development team as a Software Developer intern, developing IPC Unigy 360 application which is a cloud based communication application for traders in global financial market.


My main interests are in the field of software engineering, Internet of Things, machine learning and cloud computing. I am deeply passionate about tackling and solving problems in the field of software engineering.


In my first semester at Stony Brook, I had a fantastic opportunity to be a teaching assistant for undergraduate course ESE224 - C++,Data structures and algorithms. It was an enriching experience to interact with a class of approximately 80 students and learn a lot.


In Summer 2018, I had an opportunity to work as a Software Developer Intern for IPC Systems, Research and Development team. I was responsible for data visualization and data analytics of revenue of IPC’s product-Unigy 360. I worked extensively on Python and SQL, where I worked on data visualization and data analytics using Python libraries and stored the data in SQL and wrote stored procedures. Worked on a Python application to automate the process of dialing into conference calls from IPC’s Unigy 360 application. Developed REST Api’s using Flask and Node.js to integrate with the front end application.


I started my career at Ericsson India, working as a Network Engineer in Data Analytics where I had an opportunity to analyse large sets of Network data(3g/LTE) using Python and determine the error patterns, analyse the network behavior. I worked on SQL to store all the relevant network data which was helpful in data analysis. I designed and developed a web application which was used to log in errors in network, downsites and monitoring 3g/LTE carriers. Was also involved in optimization of in-SQL statements which were used for front end application. My experience in Ericsson helped me develop a strong foundation in Software development and data analysis.
