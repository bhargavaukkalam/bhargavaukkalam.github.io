About Me

I am a Computer Engineering graduate student at Stony Brook University. I joined this university in Fall 2017 and expect to graduate in December 2018.I am currently interning at IPC Systems in the R&D team as a Software Developer intern, developing IPC Unigy 360 application which is a cloud based communication application for traders in global financial market.


My main interests are in the field of software engineering, Internet of Things, machine learning and cloud computing. I am deeply passionate about tackling and solving problems in the field of software engineering.

Automated bulk excel related work using python and also gained experience in development of web applications which was used for logging in errors in site data.
